# 🌍 ITINERA – AI Smart Travel Planner

ITINERA is an AI-powered travel planning web application that generates personalized travel itineraries based on user preferences, budget, and accessibility needs.

## 🚀 Features

* 🎯 Personalized travel recommendations
* 💰 Budget-based itinerary generation
* ♿ Accessibility support for specially-abled users
* 🔊 Voice input & text-to-speech output for visually impaired users
* 🌦️ Weather-based activity adjustments
* 🗺️ Smart travel planning

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript, Bootstrap
* Backend: Flask (Python)
* Database: SQLite
* AI/ML: Pandas, Scikit-learn

## 📂 Project Structure

* app.py → Backend logic
* templates/ → HTML files
* static/ → CSS, JS, images
* datasets/ → Travel data

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```
